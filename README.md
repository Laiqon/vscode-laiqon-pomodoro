# Laiqon Pomodoro

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

![Laiqon Pomodoro](https://github.com/Laiqon/vscode-laiqon-pomodoro/blob/main/media/laiqon-logo-horizontal.png?raw=true)

A beautiful Pomodoro timer and an elegant interface to boost your productivity

## Features

- Customizable focus sessions (1-60 minutes)
- Short and long break support
- Background themes
- Sound alerts on phase transitions
- Status bar integration
- Rounds per session configuration
- Auto-start breaks and focus options

## Commands

- `Pomodoro: Open Pomodoro Timer` - Open the timer panel

## Configuration

| Setting | Default | Description |
|---------|---------|-------------|
| `pomodoro.focusDuration` | 25 | Focus duration in minutes |
| `pomodoro.breakDuration` | 5 | Break duration in minutes |
| `pomodoro.roundsPerSession` | 2 | Number of rounds per session |
| `pomodoro.autoStartBreaks` | true | Auto-start break after focus |
| `pomodoro.autoStartFocus` | false | Auto-start focus after break |
| `pomodoro.backgroundImage` | aurora | Background theme |
| `pomodoro.enableSounds` | true | Enable sound alerts |
